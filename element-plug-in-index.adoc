= Plug-in NetApp Element para vCenter Server
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
O plug-in NetApp Element para vCenter Server fornece uma integração com a interface do VMware vSphere, permitindo gerenciar e monitorar clusters de armazenamento que executam o software NetApp Element .

Para saber mais sobre o plug-in Element para vCenter Server, consulte o seguinte: https://docs.netapp.com/us-en/vcp/index.html["Documentação do plug-in NetApp Element para vCenter Server"^] .
