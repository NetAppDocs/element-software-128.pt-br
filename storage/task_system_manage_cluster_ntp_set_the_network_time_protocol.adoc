= Configure o Protocolo de Tempo de Rede (NTP)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
A configuração do Network Time Protocol (NTP) pode ser feita de duas maneiras: instruindo cada nó em um cluster a escutar transmissões ou instruindo cada nó a consultar um servidor NTP para obter atualizações.

O NTP é usado para sincronizar relógios em uma rede.  A conexão a um servidor NTP interno ou externo deve fazer parte da configuração inicial do cluster.
