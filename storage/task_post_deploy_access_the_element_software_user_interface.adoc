= Acesse a interface do usuário do software Element.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode acessar a interface do usuário do Element usando o endereço IP virtual de gerenciamento (MVIP) do nó primário do cluster.

Você deve garantir que os bloqueadores de pop-ups e as configurações do NoScript estejam desativados em seu navegador.

Você pode acessar a interface do usuário usando endereçamento IPv4 ou IPv6, dependendo da configuração feita durante a criação do cluster.

. Escolha uma das seguintes opções:
+
** IPv6: Digite https://[endereço MVIP IPv6] Por exemplo:
+
[listing]
----
https://[fd20:8b1e:b256:45a::1234]/
----
** IPv4: Digite https://[endereço MVIP IPv4] Por exemplo:
+
[listing]
----
https://10.123.456.789/
----


. Para DNS, insira o nome do host.
. Clique em todas as mensagens de certificado de autenticação.
