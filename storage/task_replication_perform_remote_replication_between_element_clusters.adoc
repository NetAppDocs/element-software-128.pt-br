= Realize a replicação remota entre clusters que executam o software NetApp Element .
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para clusters que executam o software Element, a replicação em tempo real permite a criação rápida de cópias remotas de dados de volume.  Você pode emparelhar um cluster de armazenamento com até quatro outros clusters de armazenamento.  Você pode replicar dados de volume de forma síncrona ou assíncrona de qualquer um dos clusters em um par de clusters para cenários de failover e failback.
