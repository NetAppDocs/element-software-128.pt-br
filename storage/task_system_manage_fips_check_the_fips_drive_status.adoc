= Verifique o status FIPS da unidade.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode verificar se o recurso de unidades FIPS está habilitado no cluster usando o software NetApp Element . `GetFeatureStatus` Método da API que mostra se o status "FIPS Drives Enabled" é verdadeiro ou falso.

. Utilizando a API do Element, verifique o recurso de unidades FIPS no cluster inserindo o seguinte comando:
+
`GetFeatureStatus`

. Analise os resultados do `GetFeatureStatus` Chamada de API.  Se o valor "Unidades FIPS ativadas" for "Verdadeiro", o recurso de unidades FIPS estará ativado.
+
[listing]
----
{"enabled": true,
"feature": "FipsDrives"
}
----
