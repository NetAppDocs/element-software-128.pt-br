= Ative o recurso de unidades FIPS
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode habilitar o recurso de unidades FIPS usando o software NetApp Element . `EnableFeature` Método da API.

A criptografia em repouso deve estar habilitada no cluster e todos os nós e unidades devem ser compatíveis com FIPS, conforme indicado quando o GetFipsReport exibir o status Pronto para todos os nós.

.Etapa
. Utilizando a API do Element, habilite o FIPS em todas as unidades digitando o seguinte:
+
`EnableFeature params: FipsDrives`
