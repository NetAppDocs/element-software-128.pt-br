= Configurar as definições do cluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode visualizar e alterar as configurações de todo o cluster e executar tarefas específicas do cluster na guia Cluster da interface do usuário do Element.

Você pode configurar definições como limite de preenchimento do cluster, acesso de suporte, criptografia em repouso, volumes virtuais, SnapMirror e cliente de transmissão NTP.
