= Visualizar sessões iSCSI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode visualizar as sessões iSCSI que estão conectadas ao cluster.  Você pode filtrar as informações para incluir apenas as sessões desejadas.

. Na interface do usuário do Element, selecione *Relatórios* > *Sessões iSCSI*.
. Para visualizar os campos de critérios de filtro, clique em *Filtrar*.
