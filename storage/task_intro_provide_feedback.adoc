= Forneça feedback
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode ajudar a melhorar a interface de usuário web do software Element e solucionar quaisquer problemas de interface usando o formulário de feedback acessível em toda a interface.

. Em qualquer página da interface do usuário do Element, clique no botão *Feedback*.
. Insira as informações relevantes nos campos Resumo e Descrição.
. Anexe quaisquer capturas de tela úteis.
. Digite um nome e um endereço de e-mail.
. Selecione a caixa de seleção para incluir dados sobre seu ambiente atual.
. Clique em *Enviar*.
