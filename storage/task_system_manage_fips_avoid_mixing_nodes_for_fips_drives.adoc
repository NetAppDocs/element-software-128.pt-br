= Evite misturar nós para unidades FIPS.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para se preparar para ativar o recurso de unidades FIPS, você deve evitar misturar nós em que alguns sejam compatíveis com unidades FIPS e outros não.

Um cluster é considerado compatível com o padrão FIPS com base nas seguintes condições:

* Todos os discos rígidos possuem certificação FIPS.
* Todos os nós são nós de unidade FIPS.
* A criptografia em repouso (EAR) está ativada.
* O recurso de unidades FIPS está ativado.  Todas as unidades e nós devem ser compatíveis com FIPS e a criptografia em repouso deve estar ativada para habilitar o recurso de unidade FIPS.
