= ListGroupSnapshots
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar `ListGroupSnapshots` Método para retornar informações sobre todos os snapshots de grupo que foram criados.



== Parâmetros

Este método possui os seguintes parâmetros de entrada:

|===
| Nome | Descrição | Tipo | Valor padrão | Obrigatório 


 a| 
ID do instantâneo do grupo
 a| 
Recuperar informações para um ID de instantâneo de grupo individual.
 a| 
inteiro
 a| 
Nenhum
 a| 
Não



 a| 
volumes
 a| 
Uma matriz de IDs de volume exclusivos para consulta.  Se você não especificar esse parâmetro, todos os snapshots de grupo no cluster serão incluídos.
 a| 
matriz volumeID
 a| 
Nenhum
 a| 
Não

|===


== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
instantâneos de grupo
 a| 
Uma lista de objetos contendo informações sobre cada instantâneo do grupo.
 a| 
xref:reference_element_api_groupsnapshot.adoc[instantâneo de grupo]variedade

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "ListGroupSnapshots",
  "params": {
  "volumes": [
     31,
     49
    ]
  },
  "id": 1
}
----
