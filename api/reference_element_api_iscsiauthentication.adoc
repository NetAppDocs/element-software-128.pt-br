= Autenticação ISCSI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O objeto ISCSIAuthentication contém informações de autenticação sobre uma sessão ISCSI.



== Membros do objeto

Este objeto contém os seguintes membros:

|===
| Nome | Descrição | Tipo 


 a| 
método de autenticação
 a| 
O método de autenticação usado durante o login da sessão iSCSI, por exemplo, CHAP ou Nenhum.
 a| 
corda



 a| 
chapAlgoritmo
 a| 
O algoritmo CHAP utilizado é, por exemplo, MD5, SHA1*, SHA-256* ou SHA3-256*.
 a| 
corda



 a| 
chapUsername
 a| 
O nome de usuário CHAP especificado pelo iniciador durante o login de uma sessão iSCSI.
 a| 
corda



 a| 
direção
 a| 
A direção da autenticação, por exemplo, unidirecional (apenas o iniciador) ou bidirecional (tanto o iniciador quanto o alvo).
 a| 
corda

|===
* Disponível a partir do Element 12.7.
