= Listar trabalhos em grande volume
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `ListBulkVolumeJobs` Método para obter informações sobre cada operação de leitura ou gravação em massa que ocorre no sistema.



== Parâmetros

Este método não possui parâmetros de entrada.



== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
trabalhos em massa
 a| 
Um conjunto de informações para cada trabalho de grande volume.
 a| 
xref:reference_element_api_bulkvolumejob.adoc[Trabalho em massa]variedade

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
   "method": "ListBulkVolumeJobs",
   "params": {
      },
   "id" : 1
}
----
