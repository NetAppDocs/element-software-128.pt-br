= domíniosDeProteçãoDeNó
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O objeto nodeProtectionDomains contém informações sobre a identidade de um nó e os domínios de proteção associados a esse nó.



== Membros do objeto

Este objeto contém os seguintes membros:

|===
| Nome | Descrição | Tipo 


 a| 
ID do nó
 a| 
Identificador único para o nó.
 a| 
inteiro



 a| 
domínios de proteção
 a| 
Lista de domínios de proteção dos quais o nó é membro.
 a| 
link:reference_element_api_protectiondomain.html["domínio de proteção"]

|===
