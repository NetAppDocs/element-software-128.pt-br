= ListProtectionDomainLevels
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `ListProtectionDomainLevels` Método para listar os níveis de tolerância e resiliência do cluster de armazenamento.  Os níveis de tolerância indicam a capacidade do cluster de continuar lendo e gravando dados em caso de falha, e os níveis de resiliência indicam a capacidade do cluster de armazenamento de se recuperar automaticamente de uma ou mais falhas.



== Parâmetro

Este método não possui parâmetro de entrada.



== Valores de retorno

Este método tem os seguintes valores de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
níveisDoDomínioDeProteção
 a| 
Uma lista dos diferentes níveis de domínio de proteção, onde cada um fornece informações sobre a tolerância e a resiliência do cluster de armazenamento.
 a| 
xref:reference_element_api_protectiondomainlevel.adoc[nívelDoDomínioDeProteção]

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "ListProtectionDomainLevels",
  "params": {},
  "id" : 1
}
----
