= Autenticação API
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode se autenticar no sistema ao usar a API incluindo um cabeçalho de autenticação HTTP Basic em todas as solicitações da API.  Se você omitir as informações de autenticação, o sistema rejeitará a solicitação não autenticada com uma resposta HTTP 401.  O sistema suporta autenticação HTTP básica sobre TLS.

Utilize a conta de administrador do cluster para autenticação da API.
