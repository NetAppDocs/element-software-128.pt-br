= GetIpmiConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `GetIpmiConfig` Método para recuperar informações de sensores de hardware presentes em seu nó.



== Parâmetro

Este método possui o seguinte parâmetro de entrada:

|===
| Nome | Descrição | Tipo 


 a| 
tipo de chassi
 a| 
Utilizado para exibir informações para cada tipo de chassi de nó. Valores possíveis:

* todos: retorna informações do sensor para cada tipo de chassi.
* {tipo de chassi}: retorna informações do sensor para um tipo de chassi especificado.

 a| 
corda

|===


== Valores de retorno

Este método tem os seguintes valores de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
nomeDoSensor
 a| 
Nome do sensor encontrado.
 a| 
corda



 a| 
ID único do sensor
 a| 
Identificador único para o sensor.
 a| 
corda

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "GetIpmiConfig",
  "params": {
     "chassisType"; "all"
     },
  "id" : 1
}
----
