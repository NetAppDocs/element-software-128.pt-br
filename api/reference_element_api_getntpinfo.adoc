= GetNtpInfo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `GetNtpInfo` Método para obter as informações de configuração atuais do protocolo de tempo de rede (NTP).



== Parâmetro

Este método não possui parâmetro de entrada.



== Valores de retorno

Este método tem os seguintes valores de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
servidores
 a| 
Lista de servidores NTP.
 a| 
matriz de strings



 a| 
cliente de transmissão
 a| 
Indica se os nós do cluster estão ou não à escuta de mensagens NTP de transmissão. Valores possíveis:

* verdadeiro
* falso

 a| 
booleano

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "GetNtpInfo",
  "params": {},
  "id" : 1
}
----


== Exemplo de resposta

Este método retorna uma resposta semelhante ao exemplo a seguir:

[listing]
----
{
  "id" : 1,
  "result" : {
    "broadcastclient" : false,
    "servers" : [ "us.pool.ntp.org" ]
   }
}
----
