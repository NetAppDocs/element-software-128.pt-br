= Listar nós ativos pendentes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `ListPendingActiveNodes` Método para listar os nós do cluster que estão no estado PendingActive, entre os estados pendente e ativo.  Os nós nesse estado estão sendo restaurados à imagem de fábrica.



== Parâmetros

Este método não possui parâmetros de entrada.



== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
nósativospendentes
 a| 
Lista de objetos que detalham informações sobre todos os nós PendingActive no sistema.
 a| 
xref:reference_element_api_pendingactivenode.adoc[nóativo pendente]variedade

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
   "method": "ListPendingActiveNodes",
   "params": {},
   "id" : 1
}
----
