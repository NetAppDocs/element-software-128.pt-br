= snapMirrorPolicyRule
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O objeto snapMirrorPolicyRule contém informações sobre as regras em uma política do SnapMirror .



== Membros do objeto

Este objeto contém os seguintes membros:

|===
| Nome | Descrição | Tipo 


 a| 
snapMirrorLabel
 a| 
A etiqueta de cópia instantânea é usada para selecionar cópias instantâneas em relações de proteção de dados estendidas.
 a| 
corda



 a| 
manterContagem
 a| 
Especifica o número máximo de cópias de instantâneo que são retidas no volume de destino do SnapMirror para uma regra.
 a| 
inteiro

|===
