= Redefinir configuração de rede
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `ResetNetworkConfig` Método para ajudar a resolver problemas de configuração de rede para um nó individual.  Este método redefine a configuração de rede de um nó individual para as configurações padrão de fábrica.



== Parâmetros

Este método não possui parâmetros de entrada.



== Valor de retorno

Este método não possui valores de retorno.



== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
   "method": "ResetNetworkConfig",
   "params": {},
   "id" : 1
}
----


== Exemplo de resposta

Este método não retorna uma resposta.

11,0
