= ObterEstatísticasDoCluster
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `GetClusterStats` Método para obter medições de atividade de alto nível para o cluster.  Os valores retornados são cumulativos desde a criação do cluster.



== Parâmetros

Este método não possui parâmetros de entrada.



== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
clusterStats
 a| 
Informações sobre atividades do cluster.
 a| 
xref:reference_element_api_clusterstats.adoc[clusterStats]

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
   "method": "GetClusterStats",
   "params": {},
   "id" : 1
}
----
