= Desativar Ssh
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode usar o `DisableSsh` Método para desativar o serviço SSH em um único nó de armazenamento.  Este método não afeta a duração do tempo limite do serviço SSH em todo o cluster.



== Parâmetro

Este método não possui parâmetro de entrada.



== Valor de retorno

Este método tem o seguinte valor de retorno:

|===
| Nome | Descrição | Tipo 


 a| 
habilitado
 a| 
O estado do serviço SSH para este nó.
 a| 
booleano

|===


== Exemplo de solicitação

As solicitações para esse método são semelhantes ao seguinte exemplo:

[listing]
----
{
  "method": "DisableSsh",
  "params": {
      },
  "id" : 1
}
----


== Exemplo de resposta

Este método retorna uma resposta semelhante ao exemplo a seguir:

[listing]
----
{
  "id" : 1,
  "result" : {"enabled": false}
}
----
